# Master Manager - TikTok Automation Hub

This repository contains the official legal documentation for the **Master Manager** application.

### Project Overview
Master Manager is an internal workflow orchestration tool built with **n8n** and **Supabase**. It serves as the central hub for managing content scheduling, affiliate link mapping, and automated publishing across a portfolio of niche-specific TikTok accounts.

### Tech Stack
* **Automation:** n8n (Node-based workflow engine)
* **Database:** Supabase (PostgreSQL for secure token and memory storage)
* **Data Source:** Google Sheets (Product inventory and scheduling)
* **API:** TikTok Content Posting API
